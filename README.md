# Snowflake Geo Demo

This project demonstrates geospatial functions in Snowflake, including constructors, accessors, relationships, measurements, transformations, output formats, and H3 functions.

## Setup

Run `setup.sql` to create the necessary tables and functions.

## Structure

- `scripts/`: Original scripts for each category.
- `new_scripts/`: Updated/modernized scripts.

For more details, refer to the individual SQL files.
